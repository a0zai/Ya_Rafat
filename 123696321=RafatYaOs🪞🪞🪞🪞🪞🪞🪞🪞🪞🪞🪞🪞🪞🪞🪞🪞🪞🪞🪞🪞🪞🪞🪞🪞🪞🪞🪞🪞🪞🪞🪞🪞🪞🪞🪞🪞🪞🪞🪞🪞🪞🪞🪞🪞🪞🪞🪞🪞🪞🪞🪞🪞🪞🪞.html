Getting Started

Include Bootstrap's CSS and JS via CDN:

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>

Customize

Adjust variables and use SASS for custom builds:

$primary: #563d7c;
$font-family-sans-serif: "Helvetica Neue", Arial;

Layout

Build responsive grids using rows and columns:

<div class="row">
  <div class="col-md-6">Column 1</div>
  <div class="col-md-6">Column 2</div>
</div>

Content

Style headings, paragraphs, and images:

<h1 class="display-4">Hello, world!</h1>

Forms

Create modern forms with validation:

<input type="email" class="form-control" placeholder="Enter email">

Components

Use buttons, alerts, navbars, and more:

<button class="btn btn-primary">Click me</button>

Helpers

Use helper classes for spacing and text:

.text-center .m-3 .p-2

Utilities

Quickly adjust layout with utility classes:

.bg-primary .text-white .d-flex

Extend

Extend Bootstrap with custom components:

import 'bootstrap/dist/css/bootstrap.css';
