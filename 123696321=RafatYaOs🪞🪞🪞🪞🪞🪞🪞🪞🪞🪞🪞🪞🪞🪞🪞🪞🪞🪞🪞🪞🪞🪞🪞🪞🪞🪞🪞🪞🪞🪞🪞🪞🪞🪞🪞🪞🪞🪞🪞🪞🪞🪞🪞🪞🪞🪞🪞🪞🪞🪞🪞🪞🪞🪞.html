Getting Started

Include Bootstrap's CSS and JS via CDN:

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>

Customize

Adjust variables and use SASS for custom builds:

$primary: #563d7c;
$font-family-sans-serif: "Helvetica Neue", Arial;

Layout

Build responsive grids using rows and columns:

<div class="row">
  <div class="col-md-6">Column 1</div>
  <div class="col-md-6">Column 2</div>
</div>

Content

Style headings, paragraphs, and images:

<h1 class="display-4">Hello, world!</h1>

Forms

Create modern forms with validation:

<input type="email" class="form-control" placeholder="Enter email">

Components

Use buttons, alerts, navbars, and more:

<button class="btn btn-primary">Click me</button>

Helpers

Use helper classes for spacing and text:

.text-center .m-3 .p-2

Utilities

Quickly adjust layout with utility classes:

.bg-primary .text-white .d-flex

Extend

Extend Bootstrap with custom components:

import 'bootstrap/dist/css/bootstrap.css';

Quick start

Get started by including Bootstrap’s production-ready CSS and JavaScript via CDN without the need for any build steps. See it in practice with this Bootstrap CodePen demo.

  1. Create a new index.html file. Include the <meta name="viewport"> tag for proper responsive behavior. 
    <!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>Bootstrap demo</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
  </body>
</html>
  2. Include Bootstrap’s CSS and JS via CDN: 
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.7/dist/css/bootstrap.min.css" rel="stylesheet">

    CDN links
    • CSS: https://cdn.jsdelivr.net/npm/bootstrap@5.3.7/dist/css/bootstrap.min.css
    • JS: https://cdn.jsdelivr.net/npm/bootstrap@5.3.7/dist/js/bootstrap.bundle.min.js

    JS components

    Components that require JavaScript and Popper:

    • Accordions (Collapse)
    • Alerts (dismiss)
    • Buttons (toggle states)
    • Carousel
    • Collapse
    • Dropdowns (needs Popper)
    • Modals
    • Navbar (Collapse and Offcanvas)
    • Navs (Tab plugin)
    • Offcanvas
    • Scrollspy
    • Toasts
    • Tooltips and popovers (needs Popper)
    Important globals
    • HTML5 doctype: <!doctype html>
    • Viewport meta: <meta name="viewport" content="width=device-width, initial-scale=1">
    • Box-sizing: border-box
    • Reboot: CSS resets and standardizations
